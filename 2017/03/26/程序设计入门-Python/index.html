编程练习

二分法求平方根：

  • 基本思想

    • 猜测一个平方根（x/2)
    • 如果猜小了，则正确平方根在猜测数字和原数字之间
    • 如果猜大了，则在0和猜测数字之间

  • 算法描述

    • input:x

    • output:√x

      • low = 0, high = x
      • guess = (low + high) / 2
      • 如果$guess^2 == x$, 则输出guess，程序结束
      • $如果guess^2 < x , 则输出low = guess；继续步骤2$
      • 如果$guess^2 > x$，则输出 $high = guess$;继续步骤2

      对于上面的程序，当x<0或x<1时将不成立
